# Data Files Directory

This directory contains sample data files, templates, and test assets for AquaForge.ai.

## Contents

- **CSV Templates**: Sample roster templates for Seton and Opponent teams
- **Test Data**: Excel files used for testing and development
- **Assets**: Images and logos (e.g., `seton_logo.png.jpg`)

## Usage

These files are referenced by:

- **Tests**: Integration tests use these files to verify upload/parsing logic
- **Documentation**: Examples in user guides
- **Development**: Quick testing of the optimization engine

## File Types

- `.csv` - Roster templates
- `.xlsx` - Excel roster files
- `.png`, `.jpg` - Image assets
