## Meet Types Legend

| Type                        | Description                                            |
| --------------------------- | ------------------------------------------------------ |
| **Time Trial**              | Non-scored practice meet, establish times              |
| **Invitational**            | Multiple teams, scored, timed finals                   |
| **Dual Meet**               | Head-to-head vs single opponent (AquaForge optimized!) |
| **Conference Championship** | VCAC conference teams only                             |
| **Regional Championship**   | Geographic region (NoVa Catholic, VISAA Division)      |
| **State Championship**      | VISAA State-wide, prelims/finals format                |
| **National**                | National Catholic championship, qualifiers only        |
| **Charity**                 | Fundraising event, non-competitive                     |
