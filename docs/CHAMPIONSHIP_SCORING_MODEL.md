## 3. Point Value Analysis

### 3.1 Where Points Are Concentrated

**VCAC 12-Place:**
- Top 6 places: 16+13+12+11+10+9 = **71 points** (76% of total)
- Bottom 6 places: 7+5+4+3+2+1 = **22 points** (24%)
- Relay 1st place (32) > Individual 1st + 2nd (29)

**VISAA 16-Place:**
- Top 8 places: 20+17+16+15+14+13+12+11 = **118 points** (76%)
- Bottom 8 places: 9+7+6+5+4+3+2+1 = **37 points** (24%)
- Relay 1st place (40) > Individual 1st + 2nd (37)

### 3.2 Strategic Implications

| Priority | Strategy                   | Reason                                                   |
| -------- | -------------------------- | -------------------------------------------------------- |
| 1        | **Relay Optimization**     | One relay win = 40 pts (VISAA) or 32 pts (VCAC)          |
| 2        | **Top 3 Finishes**         | Top 3 combined = 53 pts (VISAA) vs bottom 8 = 37 pts     |
| 3        | **Depth Stacking**         | Places 9-12 still contribute 22+ points in championships |
| 4        | **Avoid 0-Point Finishes** | Don't waste entries on swimmers outside scoring places   |

## 5. Meet-Specific Strategy Models

### 5.1 VCAC Conference Championship (12-Place)

**Configuration:**
- Scoring: 12 places
- Entry: Top 4 per team per event can score
- Format: Timed finals (no prelims)
- Exhibition: Not allowed

**Optimal Strategy Combination:**
```
Primary:   RelayMaximization + DepthStacking
Secondary: SwingEventFocus + FatigueManagement
```

**Tactical Priorities:**
1. Win 200 Medley Relay (32 pts) and 200 Free Relay (32 pts)
2. Stack events where team has 3-4 swimmers in top 12
3. Identify swing events where 1-2 place improvements yield high ROI
4. Manage fatigue: timed finals = one swim per event

**Edge Cases:**
- Only 4 swimmers per team score, so 5th+ swimmers are wasted in individual
- Deploy non-scoring swimmers to relay duty
- Diving counts as 1 of 2 individual events

### 5.2 VISAA State Championship (16-Place)

**Configuration:**
- Scoring: 16 places
- Entry: Time standard qualification
- Format: Prelims → Championship Finals (1-8) + Consolation Finals (9-16)
- Exhibition: Bonus events don't score

**Optimal Strategy Combination:**
```
Primary:   PeakPerformance + RelaySupremacy
Secondary: ConsolationHarvesting + OpponentDefense
```

**Tactical Priorities:**
1. Qualify swimmers for Championship Finals (places 1-8 = higher points)
2. Consolation Finals still worth significant points (9th = 9 pts)
3. Relay focus: 40 pts for 1st (more than ind 1st + 2nd combined)
4. Peak performance: Prelim → Final improvement is typical

**Edge Cases:**
- Consolation finals swimmers CANNOT outscore Championship finals
- Bonus swimmers (Bonus 50 Free, etc.) do NOT score
- Miss an event = DQ for rest of meet
- If 5+ swimmers qualify top 16, coach designates which 4 score

### 5.3 VISAA Division II Invitational (12-Place)

**Configuration:**
- Scoring: 12 places
- Entry: Top 4 per team
- Format: Timed finals

**Optimal Strategy Combination:**
```
Primary:   DepthMaximization + SweepOpportunities
Secondary: StateQualification + TimeDrops
```

**Tactical Priorities:**
1. Smaller D2 pool = easier top placements
2. Identify events with weak D2 competition
3. Use as state qualification opportunity
4. Balance scoring with getting qualifying times

### 5.4 VISAA JV Invitational (12-Place)

**Configuration:**
- Scoring: 12 places
- Entry: JV level swimmers
- Format: Timed finals
- **Exhibition: ALLOWED** (unique)

**Optimal Strategy Combination:**
```
Primary:   Development + TimeDrops
Secondary: FutureRelayDevelopment + ExperienceBuilding
```

**Tactical Priorities:**
1. Enter developing swimmers in exhibition for times
2. Test future varsity relay combinations
3. Balance competitive scoring with development
4. Get freshmen/sophomores championship experience

## 6. Relay Optimization Model

### 6.1 Relay Value Analysis

| Relay      | VCAC Value  | VISAA Value | Legs                     |
| ---------- | ----------- | ----------- | ------------------------ |
| 200 Medley | 32-26-24... | 40-34-32... | Back→Breast→Fly→Free     |
| 200 Free   | 32-26-24... | 40-34-32... | 4x Free (flexible order) |
| 400 Free   | 32-26-24... | 40-34-32... | 4x Free (flexible order) |

### 6.2 Relay Strategy by Type

**200 Medley Relay (Opening):**
- Sets psychological tone for meet
- Assign stroke specialists to each leg
- A Relay: Best of each stroke
- B Relay: Second-best of each stroke

**200 Free Relay (Middle):**
- Mid-meet momentum builder
- Use 4 fastest 50 free times
- Order: Fastest lead-off, second-fastest anchor

**400 Free Relay (Closing):**
- Often decides close meets
- Consider fatigue from earlier events
- Order: Strong swimmers for legs 1-2, fastest anchor

### 6.3 Relay Trade-off Analysis

When swimmer must choose between individual event and relay:

```
If relay_points_gained > individual_points_lost:
    Assign to relay
Else:
    Keep in individual event
```

**Example:**
| Swimmer | Individual Event  | Ind. Points | Relay Points Gain | Decision        |
| ------- | ----------------- | ----------- | ----------------- | --------------- |
| Mike    | 100 Free (4th=15) | -15         | +3 (faster relay) | Keep individual |
| Tom     | Non-scorer        | 0           | +2                | Use in relay    |
| Dan     | 100 Free (7th=12) | -12         | +10               | Keep individual |
